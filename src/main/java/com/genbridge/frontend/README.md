# GenBridge UI

Frontend client for GenBridge, built with Vite + React + TypeScript.

## Prerequisites

- Node.js 18+
- npm 9+

## Run locally

```sh
npm install
npm run dev
```

Default dev URL: `http://localhost:8080`.

## Build

```sh
npm run build
npm run preview
```

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
